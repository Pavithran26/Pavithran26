## 🧠 Projects

<div align="center">

<!-- PROJECT BANNER -->
<img src="https://capsule-render.vercel.app/api?type=soft&color=gradient&customColorList=6,11,20&height=80&section=header&text=🤖%20Sattam%20AI%20—%20Tamil%20Nadu%20Law%20Chatbot&fontSize=26&fontColor=fff&animation=twinkling" />

</div>

> **Postgraduate Final Year Major Project** | N.G.P Arts and Science College, Coimbatore (2026)

**Sattam AI** is a full-stack, AI-powered legal assistant specialized for Tamil Nadu laws. It ingests Tamil Nadu legal PDFs (acts, rules, regulations), processes them into a vector database using **RAG (Retrieval-Augmented Generation)**, and provides intelligent chat-based querying across web and mobile platforms.

<div align="center">

<!-- PROJECT TECH BADGES -->

<br/><br/>

<img src="https://img.shields.io/badge/📚_50%2B_Tamil_Nadu_Acts-Pre--Loaded-2EC866?style=for-the-badge" />

<br/>

**✨ Key Features**
- 🔍 **RAG Pipeline** — Sentence-transformer embeddings from legal PDFs stored in ChromaDB
- 🧠 **Multi-LLM Support** — Google Gemini (free tier) or OpenAI-compatible via Cloudflare AI Gateway
- 📄 **Document Management** — Upload & process Tamil Nadu law PDFs with web scraping support
- 💬 **Real-time Chat** — Context-aware responses on both web and mobile
- 🔐 **Auth** — Clerk integration for user management
- 📱 **Cross-Platform** — Flutter mobile app + Next.js web app
- 📚 **Data** — Pre-loaded with 50+ Tamil Nadu acts & rules

**🛠️ Tech Stack**

| Layer | Technologies |
|---|---|
| Backend | FastAPI · LangChain · ChromaDB · Python 3.10+ |
| Web Frontend | Next.js 16+ · React 19 · TailwindCSS · Clerk |
| Mobile App | Flutter 3.8+ · Dart · Clerk Flutter |
| AI / ML | Google Gemini · OpenAI · sentence-transformers |
